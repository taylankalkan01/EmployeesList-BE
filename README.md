# EmployeesList Project Backend

___
The application has 1 api.

server.port = 8081

---

1-) Employee API ("v1/employees")
-
```html
--Postman--
GET /v1/employees - retrieves all Employees
```

Employee MODEL
-

- id (long)
- firstName (String)
- lastName (String)
- email (String)

----

### Tech Stack

---
- Java 11
- Spring Boot
- Spring Data JPA and Hibernate
- Lombok
- H2 In memory database



### Prerequisites

---
- Maven
